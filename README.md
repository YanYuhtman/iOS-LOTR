# LOTR Currency Converter

## Description
This is a sample iOS application built with SwiftUI that demonstrates a currency converter theme around The Lord of the Rings universe. It allows users to convert between various fictional currencies from Middle-earth.

## Features
- Convert between different LOTR-themed currencies (e.g., Gold Pieces, Silver Pieces, Copper Pennies).
- Simple and intuitive user interface.

## Requirements
- Xcode 15.0+
- iOS 17.0+
- Swift 5.9+

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/LOTR.git
   ```
2. Open the project in Xcode:
   ```bash
   cd LOTR
   open LOTR.xcodeproj
   ```
3. Select a simulator or a physical device and run the application.

## Usage
- Select the "from" and "to" currencies using the currency selectors.
- Enter the amount you wish to convert.
- The converted amount will be displayed.
